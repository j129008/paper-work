# 斷句程式使用說明
初次使用本程式須先執行「config.sh」，方式如下：
```
./config.sh
```

以下是資料夾及其對應的論文實驗章節。

資料夾內容以及執行方法在下文分別介紹。

## 文字前處理
### 統一符號
text_preproc.py
```
./text_preproc.sh [ 原始語料路徑 ]
```
### 語料分割
text_split.sh
text_split.py
