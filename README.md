## Folders
data              : 文字檔案資料(佛寺志, 墓誌銘, 小韻表)
lib               : 斷句函式庫
ref               : 名詞表
pickles           : 存放訓練好的資料(word2vec, lstm/crf model)
Text_Preproc      : 前處理程式
Text_Classify     : 分類程式
Text2Feature      : 特徵提取
CRF_Learner       : CRF斷句程式
LSTM_Learner      : LSTM斷句程式
CRF_LSTM_Ensemble : LSTM與CRF結合的斷句程式
Other_Model       : 其他斷句程式
